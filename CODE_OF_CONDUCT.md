# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

Read the full text here: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

This Code applies to all project spaces — the GitHub repository (issues, pull requests, discussions, commits, code reviews), the YouTube comments, any social-media thread tagged with `#pulse-player`, and any in-person or online event where someone represents the project.

## Reporting

Report unacceptable behaviour by email to **`yamadaablog@gmail.com`** with subject prefix `[pulse-player conduct]`. Reports are read by the project maintainer only and stay confidential.

The maintainer commits to:

- Acknowledging every report within **72 hours**.
- Investigating in good faith.
- Communicating the outcome to the reporter.
- Not retaliating against anyone who reports in good faith.

## Enforcement

Consequences for violations follow the **Community Impact Guidelines** from the Contributor Covenant 2.1 (linked above) — ranging from a private clarification to a permanent ban depending on severity.

## Why this short page

The full text of the Contributor Covenant is well-maintained at the link above and updated periodically. Inlining it here would risk drift; linking keeps this page short and authoritative.
